Available Routes:

  • /api/v1.0/precipitation

    • returns JSON dictionary of precipitation values for each date in the dataset

  • /api/v1.0/stations

    • returns name and identifier for the stations from which measurements were taken

  • /api/v1.0/tobs

    • returns JSON list of temperature observations over the past year for the most active station

  • /api/v1.0/{start}

    and

    /api/v1.0/{start}/{end}

    • returns min, max, and average temperature observations between the specified dates ('yyyy-mm-dd') for the most active station
    • if no end date is given, the end date will be the last date in the dataset
